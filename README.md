# Подробней о проекте Антивирус Монтировка
**Антивирус Монтировка** - является **open-source** проектом на **Python3**, распространяющимся по лицензии **GPL v3.0**
Что-то мини антивируса и набора утилит для обнаружения и ликвидации вирусов с ОС Windows. Проект открыт и каждый может поучаствовать в разработке, пусть жизнь в ПК станет безопасней.
Если вы считаете, что это проект бесполезен или он сделан через жопу, то поставте **Linux** и не партесь об этом.

**Установка**
Для полноценой установки вам потребуется скомпилировать программу (это не страшно! За вас всё сделали, почти), но чтоб протестировать программу вы можете просто скачать .exe файл из этого репоризитория, он уже готов к работе.
Если вы устанавливаете программу в первый раз скачайте .zip архив из этого репозитория распакуйте (желательно не на рабочий стол потому, что в этот же каталог будет распаковано много файлов). Затем установите Python (установщик в архиве), если у вас есть python 3.12 или выше пропустите этот этап. После установки запустите install_library.bat от имени администратора (он установит нужные бибилоиотеки, а если они есть то обновит), после запускайте сам установщик от имени администратора.
Затем вас попросят ввести название программы, это будет название сиполняемого файла.
Затем в появившемся окнах выберите иконку программы (в архиве иконка есть в каталоге icon), также вы можете выбрать свою иконку в формате .ico, затем выберите архив crowbar_code....zip.
У вас откроется окно с натсройкой, внимательно читайте, что значят параметры и настройте их под себя, затем нажмите "Экспорт" чтобы сохранить их отдельно для себя (если хотите можете пропустить), нажимаете "Применить" и ждёте.
После окончания компиляции введите каталог в какой каталог переместить картинки из каталога info_image, можете ничего не вводить если вам всё равно. Затем введите каталог куда скопирвоать файл ЦЕЛИКОМ (например C:\\ProgramData\\CA.exe (лучше писать \\ или /)).
Выбираете добавлять программу в атозагрузку или нет и всё!

Мы будем рады если при возникновении ошибок вы отправите все логи программы нам! Это поможет в улучшении программы.
